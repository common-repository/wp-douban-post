== Description ==

Send one message to your douban broadcasting mysaying when you publish the post.


== Installation ==

<ol>
<li>Unzip wp-douban-post to the '/wp-content/plugins/' directory</li>
<li>Activate the plugin through the 'Plugins' menu in WordPress</li>
<li>Visit WP Douban Post Options</li>
<li>Click the button "Get Access Token", it will open a new window for you to authorize WP Douban Post.</li>
<li>Close the window after the authorization is done.</li>
<li>The option page will refresh automatically. If everything works fine, you can see you Douban ID and access token.</li>
<li>Next time when you publish the post, there will be one message sent to your Douban Broadcasting Mysaying.</li>
<li>Any problem, please contact @<a href="http://twitter.com/windlx">windlx</a> at Twitter.</li>
</ol>
